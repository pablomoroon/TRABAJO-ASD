# TRABAJO DE OPTIMIZACIÓN DE 'EL JUEGO DE LA VIDA DE CONWAY'
Proyecto realizado durante el curso 2025-2026 en la asignatura del 3º curso de la carrera Ingeniería Informática - Tecnologías Informáticas de la Universidad de Sevilla, conocida como Arquitectura de Sistemas Distribuidos.
El proyecto se basa en partir de un código secuencial y optmizarlo con las distintas tecnicas vistas en clase tales como 02, 03, Instrucciones SIMD, OPENMP, MPI o desenrollado.
En el repositorio se adjuntan los distintos códigos y gráficas que nos permiten comparar los tiempos de las distintas versiones.

Trabajo realizado en colaboración de Juan Manuel Vargas Bellido
